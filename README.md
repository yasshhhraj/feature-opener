# Feature Opener

Open, copy and manage layered backend feature files side by side.

## Commands

- **Open Imported Files** — opens all files in the import chain of the active file side by side
- **Copy Feature Chain** — copies the entire import chain with a new feature name
- **Close Related Files** — closes all tabs except the active one

## How to use

1. Open any file in your feature chain (e.g. `user.controller.ts`)
2. Open the Command Palette (`Ctrl+Shift+P`)
3. Run one of the Feature Opener commands

## Requirements

Works best with layered backend projects (controllers, services, repositories).
